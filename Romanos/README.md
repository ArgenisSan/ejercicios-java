## Instrucciones de uso

1. Compilar el archivo Romanos.java desde la consola de java. Ej. "javac Romanos.java".

2. Una vez compilado, ejecutar el programa. Ej. "java Romanos".

3. El programa solicita entrada de numeros enteros. Ej. "456".

4. El programa realiza la conversión a romanos e imprime el resultado. Ej. "CDLVI".

5. Una vez realidad la conversion, el programa finaliza.
