## Instrucciones de uso

1. Compilar el archivo MainMorse.java desde la consola de java. Ej. "javac MainMorse.java"

2. Ejecutar el programa MainMorse.java Ej. "java MainMorse"

3. El programa solicita una entrada de texto. Ej. "hola mundo"

4. El programa realiza la conversión a codigo morse. Ej ".... --- .-.. .-     -- ..- -. -.. ---"

5. Al realizar la conversion, el programa finaliza
